# Treemap Chart

트리맵 차트는 계층적 데이터를 중첩된 사각형으로 표시하며, 각 사각형의 면적은 해당 값에 비례합니다. 전체 안에서 데이터 카테고리의 상대적 크기를 시각화하는 데 탁월합니다.

## 인터페이스

### TreemapChart Props

```typescript
interface TreemapChartProps {
  // 필수 속성
  data: TreemapData;

  // 선택 속성
  custom?: Partial<TreemapCustom>;
  title?: string;
  colors?: string[];       // 커스텀 색상 팔레트
}
```

### 데이터 구조

```typescript
type TreemapNode = {
  label: string;       // 노드 라벨
  value: number;       // 노드 값 (면적 결정)
  color?: string;      // 선택적 커스텀 색상
};

type TreemapData = {
  nodes: TreemapNode[];   // 데이터 노드 배열
  title?: string;         // 차트 제목
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { TreemapChart } from '@meursyphus/headless-chart';

const data = {
  nodes: [
    { label: '아시아', value: 4641 },
    { label: '아프리카', value: 1340 },
    { label: '유럽', value: 747 },
    { label: '북아메리카', value: 579 },
    { label: '남아메리카', value: 422 },
    { label: '오세아니아', value: 43 }
  ],
  title: '대륙별 세계 인구'
};

function BasicTreemapChart() {
  return (
    <Widget
      width="600px"
      height="400px"
      widget={TreemapChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Treemap Chart는 다음과 같은 계층 구조로 구성됩니다:

```
TreemapChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Legend (범례)
    │   └── LegendItem (범례 항목)
    └── Treemap (트리맵 영역)
        └── Node (개별 사각형)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, legend, treemap }` | 전체 차트 레이아웃 |
| `title` | `{ name }` | 차트 제목 |
| `legend` | `{ items }` | 범례 컨테이너 |
| `legendItem` | `{ label, color, index }` | 개별 범례 항목 |
| `treemap` | `{ nodes }` | 트리맵 컨테이너 |
| `node` | `{ label, value, color, index, ratio, x, y, width, height }` | 개별 사각형 노드 |

## 실제 사용 예제

### 1. 라벨이 있는 커스텀 노드

```javascript
const labeledTreemap = TreemapChart({
  data,
  custom: {
    node: ({ label, value, color, ratio, width, height }) => {
      return Container({
        decoration: new BoxDecoration({
          color: color,
          border: Border.all({ color: '#ffffff', width: 2 })
        }),
        child: Center({
          child: Column({
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Text(label, {
                style: new TextStyle({ fontSize: 14, fontWeight: 'bold', color: '#ffffff' })
              }),
              Text(`${value}`, {
                style: new TextStyle({ fontSize: 12, color: 'rgba(255,255,255,0.8)' })
              })
            ]
          })
        })
      });
    }
  }
});
```

### 2. 커스텀 색상 스키마

```javascript
const coloredTreemap = TreemapChart({
  data,
  colors: ['#3b82f6', '#ef4444', '#10b981', '#f59e0b', '#8b5cf6', '#ec4899']
});
```
