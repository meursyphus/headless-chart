# Gauge Chart

게이지 차트는 정의된 범위 내에서 단일 값을 속도계처럼 표시합니다. 진행 상황, KPI 또는 알려진 최소값과 최대값 사이에 있는 모든 지표를 표시하는 데 완벽합니다.

## 인터페이스

### GaugeChart Props

```typescript
interface GaugeChartProps {
  // 필수 속성
  data: GaugeChartData;

  // 선택 속성
  custom?: Partial<GaugeChartCustom>;
  title?: string;
}
```

### 데이터 구조

```typescript
type GaugeChartZone = {
  min: number;       // 구간 시작 값
  max: number;       // 구간 끝 값
  color: string;     // 구간 색상
};

type GaugeChartData = {
  value: number;         // 현재 값
  min?: number;          // 최소 스케일 값
  max?: number;          // 최대 스케일 값
  title?: string;        // 차트 제목
  zones?: GaugeChartZone[];  // 색상 구간
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { GaugeChart } from '@meursyphus/headless-chart';

const data = {
  value: 72,
  min: 0,
  max: 100,
  title: '성능 점수',
  zones: [
    { min: 0, max: 40, color: '#ef4444' },
    { min: 40, max: 70, color: '#f59e0b' },
    { min: 70, max: 100, color: '#10b981' }
  ]
};

function BasicGaugeChart() {
  return (
    <Widget
      width="400px"
      height="300px"
      widget={GaugeChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Gauge Chart는 다음과 같은 계층 구조로 구성됩니다:

```
GaugeChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Gauge (게이지 영역)
    │   ├── Scale (구간이 있는 호형 스케일)
    │   └── Needle (지시 바늘)
    └── ValueLabel (현재 값 표시)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, gauge, valueLabel }` | 전체 차트 레이아웃 |
| `title` | `{ name }` | 차트 제목 |
| `gauge` | `{ needle, scale }` | 게이지 컨테이너 |
| `needle` | `{ value, ratio }` | 지시 바늘 |
| `valueLabel` | `{ value, min, max }` | 현재 값 표시 |
| `scale` | `{ min, max, zones }` | 색상 구간이 있는 호형 스케일 |

## 실제 사용 예제

### 1. 커스텀 값 라벨

```javascript
const customGauge = GaugeChart({
  data,
  custom: {
    valueLabel: ({ value, min, max }) => {
      const percentage = ((value - min) / (max - min)) * 100;
      return Column({
        children: [
          Text(`${value}`, {
            style: new TextStyle({ fontSize: 32, fontWeight: 'bold', color: '#1f2937' })
          }),
          Text(`${percentage.toFixed(0)}%`, {
            style: new TextStyle({ fontSize: 16, color: '#6b7280' })
          })
        ]
      });
    }
  }
});
```

### 2. 커스텀 구간

```javascript
const zoneGauge = GaugeChart({
  data: {
    value: 85,
    min: 0,
    max: 100,
    zones: [
      { min: 0, max: 25, color: '#ef4444' },
      { min: 25, max: 50, color: '#f97316' },
      { min: 50, max: 75, color: '#eab308' },
      { min: 75, max: 100, color: '#22c55e' }
    ]
  }
});
```
