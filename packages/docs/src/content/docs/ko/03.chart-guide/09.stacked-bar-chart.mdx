# Stacked Bar Chart

누적 막대 차트는 각 막대 내에서 여러 데이터 시리즈를 겹쳐서 표시합니다. 각 카테고리의 총 값과 구성을 동시에 보여주는 데 이상적입니다.

## 인터페이스

### StackedBarChart Props

```typescript
interface StackedBarChartProps {
  // 필수 속성
  data: StackedBarChartData;

  // 선택 속성
  custom?: Partial<StackedBarChartCustom>;
  title?: string;
  direction?: 'vertical' | 'horizontal';
  getScale?: (data: StackedBarChartData) => StackedBarChartScale;
}
```

### 데이터 구조

```typescript
type StackedBarChartData = {
  labels: string[];                                  // 카테고리 라벨
  datasets: { legend: string; values: number[] }[];  // 데이터 시리즈
};

type StackedBarChartScale = {
  min: number;     // 최소값
  max: number;     // 최대값
  step: number;    // 단계 간격
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { StackedBarChart } from '@meursyphus/headless-chart';

const data = {
  labels: ['1분기', '2분기', '3분기', '4분기'],
  datasets: [
    { legend: '제품 A', values: [120, 150, 180, 200] },
    { legend: '제품 B', values: [80, 100, 90, 110] },
    { legend: '제품 C', values: [60, 70, 85, 95] }
  ]
};

function BasicStackedBarChart() {
  return (
    <Widget
      width="600px"
      height="400px"
      widget={StackedBarChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Stacked Bar Chart는 다음과 같은 계층 구조로 구성됩니다:

```
StackedBarChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Legend (범례 항목)
    └── Plot (플롯 영역)
        ├── XAxis (X축)
        ├── YAxis (Y축)
        ├── Grid (그리드 라인)
        └── Series (데이터 시리즈)
            └── BarGroup (그룹화된 누적 막대)
                └── Bar (개별 막대 세그먼트)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, legends, plot }` | 전체 차트 레이아웃 |
| `plot` | `{ xAxis, yAxis, series, grid }` | 플롯 영역 |
| `series` | `{ barGroups }` | 데이터 시리즈 컨테이너 |
| `barGroup` | `{ bars, index, label, values }` | 그룹화된 누적 막대 |
| `bar` | `{ value, label, legend, index }` | 개별 막대 세그먼트 |
| `legend` | `{ name, index }` | 범례 항목 |
| `title` | `{ name }` | 차트 제목 |
| `dataLabel` | `{ value, label, legend }` | 데이터 라벨 |
| `xAxis` | `{ line, labels, tick }` | X축 구조 |
| `yAxis` | `{ line, labels, tick }` | Y축 구조 |
| `xAxisLabel` | `{ name, index }` | X축 라벨 |
| `yAxisLabel` | `{ name, index }` | Y축 라벨 |
| `xAxisLine` | - | X축 라인 |
| `yAxisLine` | - | Y축 라인 |
| `xAxisTick` | - | X축 틱 마크 |
| `yAxisTick` | - | Y축 틱 마크 |
| `grid` | `{ xLine, yLine }` | 그리드 컨테이너 |
| `gridXLine` | - | 수직 그리드 라인 |
| `gridYLine` | - | 수평 그리드 라인 |

## 실제 사용 예제

### 1. 수평 누적 막대

```javascript
const horizontalStacked = StackedBarChart({
  data,
  direction: 'horizontal',
  custom: {
    bar: ({ value, label, legend, index }) => {
      // 커스텀 수평 막대 렌더링
    }
  }
});
```

### 2. 커스텀 막대 색상

```javascript
const coloredBars = StackedBarChart({
  data,
  custom: {
    bar: ({ value, label, legend, index }, config) => {
      const colors = { '제품 A': '#3b82f6', '제품 B': '#ef4444', '제품 C': '#10b981' };
      return Container({
        decoration: new BoxDecoration({
          color: colors[legend]
        })
      });
    }
  }
});
```
