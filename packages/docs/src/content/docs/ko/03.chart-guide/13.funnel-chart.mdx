# Funnel Chart

퍼널 차트는 순차적인 단계를 거치며 흐르는 데이터를 시각화하며, 각 단계는 일반적으로 이전 단계보다 작습니다. 영업 파이프라인, 전환 프로세스, 사용자 흐름 분석에 일반적으로 사용됩니다.

## 인터페이스

### FunnelChart Props

```typescript
interface FunnelChartProps {
  // 필수 속성
  data: FunnelChartData;

  // 선택 속성
  custom?: Partial<FunnelChartCustom>;
  title?: string;
}
```

### 데이터 구조

```typescript
type FunnelChartData = {
  stages: {
    label: string;       // 단계 라벨
    value: number;       // 단계 값
    color?: string;      // 선택적 커스텀 색상
  }[];
  title?: string;        // 차트 제목
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { FunnelChart } from '@meursyphus/headless-chart';

const data = {
  stages: [
    { label: '방문자', value: 10000 },
    { label: '리드', value: 5000 },
    { label: '적격 리드', value: 2500 },
    { label: '제안', value: 1200 },
    { label: '계약 완료', value: 600 }
  ],
  title: '영업 퍼널'
};

function BasicFunnelChart() {
  return (
    <Widget
      width="500px"
      height="400px"
      widget={FunnelChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Funnel Chart는 다음과 같은 계층 구조로 구성됩니다:

```
FunnelChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Legend (범례 항목)
    └── Funnel (퍼널 영역)
        └── Stage (개별 단계)
            ├── StageLabel (단계 이름)
            └── DataLabel (값/백분율)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, legends, funnel }` | 전체 차트 레이아웃 |
| `funnel` | `{ stages }` | 퍼널 컨테이너 |
| `stage` | `{ index, label, value, ratio, color, stageLabel, dataLabel }` | 개별 퍼널 단계 |
| `stageLabel` | `{ label, index }` | 단계 이름 라벨 |
| `dataLabel` | `{ value, percentage, label, index }` | 데이터 값/백분율 라벨 |
| `legend` | `{ label, color, index }` | 범례 항목 |
| `title` | `{ name }` | 차트 제목 |

## 실제 사용 예제

### 1. 전환율이 있는 커스텀 단계

```javascript
const conversionFunnel = FunnelChart({
  data,
  custom: {
    dataLabel: ({ value, percentage, label, index }) => {
      return Row({
        children: [
          Text(`${value.toLocaleString()}`, {
            style: new TextStyle({ fontSize: 14, fontWeight: 'bold', color: '#1f2937' })
          }),
          SizedBox({ width: 8 }),
          Text(`(${percentage.toFixed(1)}%)`, {
            style: new TextStyle({ fontSize: 12, color: '#6b7280' })
          })
        ]
      });
    }
  }
});
```

### 2. 커스텀 단계 색상

```javascript
const coloredFunnel = FunnelChart({
  data: {
    stages: [
      { label: '인지', value: 10000, color: '#3b82f6' },
      { label: '관심', value: 6000, color: '#6366f1' },
      { label: '결정', value: 3000, color: '#8b5cf6' },
      { label: '행동', value: 1500, color: '#a855f7' }
    ]
  }
});
```
