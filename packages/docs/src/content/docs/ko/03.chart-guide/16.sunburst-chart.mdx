# Sunburst Chart

선버스트 차트는 계층적 데이터를 동심원 링으로 표시하며, 각 링은 계층의 한 레벨을 나타냅니다. 파이 차트를 확장하여 부모-자식 관계를 보여주므로 다단계 카테고리 분류를 시각화하는 데 이상적입니다.

## 인터페이스

### SunburstChart Props

```typescript
interface SunburstChartProps {
  // 필수 속성
  data: SunburstChartData;

  // 선택 속성
  custom?: Partial<SunburstCustom>;
  title?: string;
}
```

### 데이터 구조

```typescript
type SunburstNode = {
  label: string;               // 노드 라벨
  value?: number;              // 노드 값 (리프 노드)
  color?: string;              // 선택적 커스텀 색상
  children?: SunburstNode[];   // 자식 노드
};

type SunburstChartData = {
  root: SunburstNode;    // 계층 구조의 루트 노드
  title?: string;        // 차트 제목
};

type FlatSegment = {
  node: SunburstNode;    // 노드 참조
  depth: number;         // 계층 내 깊이 레벨
  startAngle: number;    // 세그먼트 시작 각도
  endAngle: number;      // 세그먼트 끝 각도
  color: string;         // 세그먼트 색상
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { SunburstChart } from '@meursyphus/headless-chart';

const data = {
  root: {
    label: '전체',
    children: [
      {
        label: '카테고리 A',
        children: [
          { label: 'A-1', value: 30 },
          { label: 'A-2', value: 20 }
        ]
      },
      {
        label: '카테고리 B',
        children: [
          { label: 'B-1', value: 25 },
          { label: 'B-2', value: 15 },
          { label: 'B-3', value: 10 }
        ]
      }
    ]
  },
  title: '계층적 데이터'
};

function BasicSunburstChart() {
  return (
    <Widget
      width="500px"
      height="500px"
      widget={SunburstChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Sunburst Chart는 다음과 같은 계층 구조로 구성됩니다:

```
SunburstChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Legend (범례)
    │   └── LegendItem (범례 항목)
    └── Sunburst (선버스트 영역)
        └── Segment (개별 호 세그먼트)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, sunburst, legend }` | 전체 차트 레이아웃 |
| `title` | `{ name }` | 차트 제목 |
| `legend` | `{ items }` | 범례 컨테이너 |
| `legendItem` | `{ label, color }` | 개별 범례 항목 |
| `sunburst` | `{ segments }` | 선버스트 컨테이너 (FlatSegment[] 수신) |
| `segment` | `{ segment }` | 개별 호 세그먼트 (FlatSegment 수신) |

## 실제 사용 예제

### 1. 깊이별 커스텀 세그먼트 색상

```javascript
const depthColoredSunburst = SunburstChart({
  data,
  custom: {
    segment: ({ segment }) => {
      const depthColors = ['#3b82f6', '#60a5fa', '#93c5fd'];
      const color = depthColors[segment.depth] || '#bfdbfe';
      // 깊이 기반 색상의 커스텀 호 세그먼트 렌더링
    }
  }
});
```

### 2. 커스텀 범례

```javascript
const legendedSunburst = SunburstChart({
  data,
  custom: {
    legendItem: ({ label, color }) => {
      return Row({
        children: [
          Container({
            width: 12,
            height: 12,
            decoration: new BoxDecoration({ color, borderRadius: BorderRadius.circular(2) })
          }),
          SizedBox({ width: 6 }),
          Text(label, { style: new TextStyle({ fontSize: 12, color: '#374151' }) })
        ]
      });
    }
  }
});
```
