# Radar Chart

레이더 차트(스파이더 차트 또는 웹 차트라고도 함)는 같은 중심점에서 시작하는 축 위에 다변량 데이터를 표시합니다. 여러 차원에 걸쳐 여러 데이터셋을 동시에 비교하는 데 탁월합니다.

## 인터페이스

### RadarChart Props

```typescript
interface RadarChartProps {
  // 필수 속성
  data: RadarChartData;

  // 선택 속성
  custom?: Partial<RadarChartCustom>;
  title?: string;
  getScale?: (data: RadarChartData) => RadarChartScale;
}
```

### 데이터 구조

```typescript
type RadarChartData = {
  labels: string[];                                  // 축 라벨
  datasets: { legend: string; values: number[] }[];  // 데이터 시리즈
  title?: string;                                    // 차트 제목
};

type RadarChartScale = {
  min: number;     // 최소값
  max: number;     // 최대값
  step: number;    // 단계 간격
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { RadarChart } from '@meursyphus/headless-chart';

const data = {
  labels: ['공격력', '방어력', '속도', '체력', '마법', '지구력'],
  datasets: [
    { legend: '전사', values: [90, 85, 60, 80, 30, 75] },
    { legend: '마법사', values: [40, 50, 70, 60, 95, 55] }
  ],
  title: '캐릭터 스탯'
};

function BasicRadarChart() {
  return (
    <Widget
      width="500px"
      height="500px"
      widget={RadarChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Radar Chart는 다음과 같은 계층 구조로 구성됩니다:

```
RadarChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Legend (범례 항목)
    └── Radar (레이더 영역)
        ├── Grid (동심원 그리드 레벨)
        ├── Axes (축 라인)
        ├── AxisLabels (축 라벨)
        └── Datasets (데이터 시리즈 다각형)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, legends, radar }` | 전체 차트 레이아웃 |
| `radar` | `{ grid, axes, datasets, axisLabels }` | 레이더 컨테이너 |
| `grid` | `{ levels }` | 동심원 그리드 라인 |
| `axis` | `{ index, label }` | 개별 축 라인 |
| `axisLabel` | `{ index, label }` | 축 라벨 텍스트 |
| `dataset` | `{ values, legend, index }` | 데이터 시리즈 다각형 |
| `legend` | `{ name, index }` | 범례 항목 |
| `title` | `{ name }` | 차트 제목 |

```javascript
const styledRadar = RadarChart({
  data,
  custom: {
    dataset: ({ values, legend, index }, config) => {
      const colors = ['rgba(59, 130, 246, 0.3)', 'rgba(239, 68, 68, 0.3)'];
      const borderColors = ['#3b82f6', '#ef4444'];
      // 채우기와 스트로크가 있는 커스텀 다각형 렌더링
    }
  }
});
```

### 2. 커스텀 그리드

```javascript
const customGrid = RadarChart({

      // 커스텀 동심 도형 그리기 (원 또는 다각형)
    }
  }

```
