# Stacked Area Chart

누적 영역 차트는 여러 데이터 시리즈를 서로 겹쳐진 채워진 영역으로 표시합니다. 시간의 흐름이나 카테고리 전체에 걸쳐 개별 부분이 누적 합계에 어떻게 기여하는지 보여주는 데 이상적입니다.

## 인터페이스

### StackedAreaChart Props

```typescript
interface StackedAreaChartProps {
  // 필수 속성
  data: StackedAreaChartData;

  // 선택 속성
  custom?: Partial<StackedAreaChartCustom>;
  title?: string;
  getScale?: (data: StackedAreaChartData) => StackedAreaChartScale;
}
```

### 데이터 구조

```typescript
type StackedAreaChartData = {
  labels: string[];                                  // 카테고리 라벨
  datasets: { legend: string; values: number[] }[];  // 데이터 시리즈
};

type StackedAreaChartScale = {
  min: number;     // 최소값
  max: number;     // 최대값
  step: number;    // 단계 간격
};
```

## 기본 사용법

```javascript
import Widget from '@meursyphus/flitter-react';
import { StackedAreaChart } from '@meursyphus/headless-chart';

const data = {
  labels: ['1월', '2월', '3월', '4월', '5월', '6월'],
  datasets: [
    { legend: '데스크톱', values: [500, 520, 530, 510, 540, 560] },
    { legend: '모바일', values: [200, 250, 300, 350, 400, 450] },
    { legend: '태블릿', values: [50, 60, 70, 80, 90, 100] }
  ]
};

function BasicStackedAreaChart() {
  return (
    <Widget
      width="600px"
      height="400px"
      widget={StackedAreaChart({ data })}
    />
  );
}
```

## 차트 구성 요소

Stacked Area Chart는 다음과 같은 계층 구조로 구성됩니다:

```
StackedAreaChart
└── Layout (전체 레이아웃)
    ├── Title (차트 제목)
    ├── Legend (범례 항목)
    └── Plot (플롯 영역)
        ├── XAxis (X축)
        ├── YAxis (Y축)
        ├── Grid (그리드 라인)
        └── Series (데이터 시리즈)
            └── Area (개별 누적 영역)
```

## 커스터마이징 가능한 요소

`custom` prop을 통해 다음 구성 요소를 커스터마이징할 수 있습니다:

| 컴포넌트 | 인자 | 설명 |
|----------|------|------|
| `layout` | `{ title, legends, plot }` | 전체 차트 레이아웃 |
| `plot` | `{ xAxis, yAxis, series, grid }` | 플롯 영역 |
| `series` | `{ areas }` | 데이터 시리즈 컨테이너 |
| `area` | `{ values, cumulativeValues, legend, index }` | 개별 누적 영역 |
| `legend` | `{ name, index }` | 범례 항목 |
| `title` | `{ name }` | 차트 제목 |
| `dataLabel` | `{ value, label, legend }` | 데이터 라벨 |
| `xAxis` | `{ line, labels, tick }` | X축 구조 |
| `yAxis` | `{ line, labels, tick }` | Y축 구조 |
| `xAxisLabel` | `{ name, index }` | X축 라벨 |
| `yAxisLabel` | `{ name, index }` | Y축 라벨 |
| `xAxisLine` | - | X축 라인 |
| `yAxisLine` | - | Y축 라인 |
| `xAxisTick` | - | X축 틱 마크 |
| `yAxisTick` | - | Y축 틱 마크 |
| `grid` | `{ xLine, yLine }` | 그리드 컨테이너 |
| `gridXLine` | - | 수직 그리드 라인 |
| `gridYLine` | - | 수평 그리드 라인 |

## 실제 사용 예제

### 1. 커스텀 영역 색상

```javascript
const coloredAreas = StackedAreaChart({
  data,
  custom: {
    area: ({ values, cumulativeValues, legend, index }) => {
      const colors = ['rgba(59, 130, 246, 0.6)', 'rgba(16, 185, 129, 0.6)', 'rgba(245, 158, 11, 0.6)'];
      // 그래디언트 채우기가 있는 커스텀 영역 렌더링
    }
  }
});
```

### 2. 커스텀 데이터 라벨

```javascript
const labeledAreas = StackedAreaChart({
  data,
  custom: {
    dataLabel: ({ value, label, legend }) => {
      return Text(`${value}`, {
        style: new TextStyle({
          fontSize: 10,
          color: '#374151'
        })
      });
    }
  }
});
```
